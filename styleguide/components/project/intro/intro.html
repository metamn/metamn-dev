Description

This is the styleguide accompannying the site Metamn. It serves as a visual documentation for the framework and components building the site.

Any changes made to the site design is automatically reflected in this styleguide.

We use the BEM methodology to name and structure code. BEM is using Framework for basic components like typography, colors, the grid and Project for all other components specific to the site like headers, thumbs, portfolio sections.
