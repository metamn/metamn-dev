Styleguide Text Style

Text Style

Description

Typographic settings for different text types. Each text type has set a font, a scale, a color and additional parameters like letter-spacing, max-width or else. Everything text is set using this text-style mixin. We don't set color or font directly for a text.

Default

This is the default text. With a high contrast background and a modern typeface with extra letter spacing it should look electric, vibrant, energizing on all displays.

Body text

The body text—used for long text blocks with more than one paragraphs like the work description or the about section—is set to be highly readable.

Every line displays around 35-40 characters only and paragraphs are separated with an empty line.

Title

A page or section title. Bold.

Menu

Modern

This website wanted to be very simple and clean yet powerful.

The look and feel was designed to be electric, vibrant, energizing and modern.

For that the principles of modern design (Bauhaus) where used. Everything was build from bottom up. New elements use only a minimum necessary deviation from a standard base.

These typographic settings should reflect this approach. Different text styles must differ minimally from each other yet still serve their function.
